# 3D-Reversi 仕様書

## 概要

シングル HTML ファイルで動作する WebGL ベースの 3D リバーシゲーム。
外部ライブラリへの依存なし。Pure WebGL + Vanilla JS で実装する。

---

## ゲームルール

### 基本ルール（標準リバーシ準拠）

- 8×8 のボードで対戦（2人プレイ または vs CPU）
- 黒が先手、白が後手
- 自分の石で相手の石を挟むと、挟まれた石が自分の石に変わる
- 縦・横・斜め 8方向への挟み込みが有効
- 着手可能なマスがない場合はパス
- 両者ともに着手不能になったらゲーム終了、石の数が多い方の勝ち

### 3D 演出ルール

- 石がひっくり返る際に 3D フリップアニメーションを再生
- 石の配置時にドロップアニメーションを再生
- ボードを自由視点で回転・ズーム可能

## 3D 描画仕様

### カメラ

- 初期位置: ボード上方 45° 俯瞰
- 操作:
  - **左ドラッグ**: 視点回転（オービット）
  - **右ドラッグ / 二本指ドラッグ**: 視点パン
  - **スクロール / ピンチ**: ズームイン・アウト
- 制限: 真下・真上方向への回転は制限（ジンバルロック回避）
- スムーズ慣性: ドラッグ離し後にゆるやかに減速

### ボード

- 8×8 グリッドの平面メッシュ
- マス目はライン描画またはテクスチャ
- ボードの色: 深緑（#1a5c2a）と薄緑（#2d8a3e）の市松模様
- ボード枠: 木目調の色（#8B4513）
- ボードの厚み: 3D 感を出すため若干の厚みを持たせる

### 石

- **形状**: 扁平な円柱（cylinder）
  - 半径: マス幅の約 40%
  - 高さ: 半径の約 30%
  - 上下面を丸くベベル処理（任意）
- **マテリアル**:
  - 黒石: 鏡面反射あり、ダークグレー〜ブラック
  - 白石: 鏡面反射あり、ホワイト〜アイボリー
- **ライティング**: Phong/Blinn-Phong シェーディング
  - 環境光 + 平行光源（太陽光相当）
  - 石表面に光沢ハイライト

### アニメーション

| アニメーション | 説明 | 時間 |
|----------------|------|------|
| 石の配置 | 上方からドロップしてバウンド | 300ms |
| 石のフリップ | Y 軸回転 180° でひっくり返る | 400ms/枚 |
| 複数フリップ | ひっくり返りを連鎖的にウェーブ演出 | 50ms ずつ遅延 |
| 配置可能ヒント | 半透明の石がゆっくり上下に浮遊 | ループ |
| ゲーム終了 | 勝利石が一斉にジャンプ | 600ms |

### シェーダー

```glsl
// 頂点シェーダー: MVP変換、法線変換
// フラグメントシェーダー: Blinn-Phong ライティング
//   - ambient + diffuse + specular
//   - 石の色は uniform で制御
```

## ゲームモード

### 対戦モード

| モード | 説明 |
|--------|------|
| 1P vs CPU | プレイヤー（黒）対 CPU（白） |
| 2P ローカル | 同一端末で2人対戦 |

### CPU 難易度

| 難易度 | アルゴリズム |
|--------|-------------|
| Easy   | ランダム着手 |
| Normal | Minimax 深さ3 |
| Hard   | Minimax + αβ 枝刈り 深さ6 |

### CPU 評価関数

- 石数差
- 角・辺の重みボーナス（位置評価テーブル）
- 着手可能数（mobility）

## 入力・操作

### マウス操作

| 操作 | 動作 |
|------|------|
| 左クリック（マス） | 石を配置 |
| 左ドラッグ（ボード外） | 視点回転 |
| 右ドラッグ | 視点パン |
| ホイール | ズーム |

### タッチ操作

| 操作 | 動作 |
|------|------|
| タップ（マス） | 石を配置 |
| 一本指ドラッグ | 視点回転 |
| 二本指ドラッグ | 視点パン |
| ピンチ | ズーム |

### キーボード

| キー | 動作 |
|------|------|
| R | カメラリセット |
| H | ヒント表示 ON/OFF |
| N | 新規ゲーム |

## 技術スタック

| 項目 | 内容 |
|------|------|
| 実装形式 | シングル HTML ファイル（.html） |
| 描画 API | WebGL 1.0（広範な互換性） |
| JavaScript | Vanilla JS (ES6+)、モジュールなし |
| CSS | インライン `<style>` タグ |
| 外部依存 | **なし** |
| 対応ブラウザ | Chrome, Firefox, Safari, Edge（最新版） |

### ファイル構成（単一 HTML 内）

```
index.html
├── <style>          UI スタイル
├── <canvas>         WebGL 描画領域
├── <div>            UI オーバーレイ（スコア、ボタン等）
└── <script>
    ├── WebGL 初期化
    ├── シェーダー（GLSL ソース文字列）
    ├── メッシュ生成（ボード、石）
    ├── レンダリングループ
    ├── アニメーションシステム
    ├── カメラ制御
    ├── ゲームロジック（リバーシルール）
    ├── CPU AI（Minimax）
    └── 入力ハンドラ（マウス、タッチ、キーボード）
```

## 非機能要件

| 項目 | 目標値 |
|------|--------|
| フレームレート | 60 FPS（アニメーション中も維持） |
| 初回ロード | 1ファイル、< 200KB（非圧縮） |
| CPU 思考時間 | Hard でも < 2秒 |
| モバイル対応 | スマートフォン・タブレットで操作可能 |
| アクセシビリティ | ターン・スコアをテキストで表示（スクリーンリーダー向け） |
