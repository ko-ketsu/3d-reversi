# 3D-Reversi 仕様書

## 概要

シングル HTML ファイルで動作する WebGL ベースの 3D リバーシゲーム。
外部ライブラリへの依存なし。Pure WebGL + Vanilla JS で実装する。

---

## ゲームルール

### 基本ルール（標準リバーシ準拠）

- 8×8 のボードで対戦（2人プレイ または vs CPU）
- 黒が先手、白が後手
- 自分の石で相手の石を挟むと、挟まれた石が自分の石に変わる
- 縦・横・斜め 8方向への挟み込みが有効
- 着手可能なマスがない場合はパス
- 両者ともに着手不能になったらゲーム終了、石の数が多い方の勝ち

### 3D 演出ルール

- 石がひっくり返る際に 3D フリップアニメーションを再生
- 石の配置時にドロップアニメーションを再生
- ボードを自由視点で回転・ズーム可能

## 3D 描画仕様

### カメラ

- 初期位置: ボード上方 45° 俯瞰
- 操作:
  - **左ドラッグ**: 視点回転（オービット）
  - **右ドラッグ / 二本指ドラッグ**: 視点パン
  - **スクロール / ピンチ**: ズームイン・アウト
- 制限: 真下・真上方向への回転は制限（ジンバルロック回避）
- スムーズ慣性: ドラッグ離し後にゆるやかに減速

### ボード

- 8×8 グリッドの平面メッシュ
- マス目はライン描画またはテクスチャ
- ボードの色: 深緑（#1a5c2a）と薄緑（#2d8a3e）の市松模様
- ボード枠: 木目調の色（#8B4513）
- ボードの厚み: 3D 感を出すため若干の厚みを持たせる

### 石

- **形状**: 扁平な円柱（cylinder）
  - 半径: マス幅の約 40%
  - 高さ: 半径の約 30%
  - 上下面を丸くベベル処理（任意）
- **マテリアル**:
  - 黒石: 鏡面反射あり、ダークグレー〜ブラック
  - 白石: 鏡面反射あり、ホワイト〜アイボリー
- **ライティング**: Phong/Blinn-Phong シェーディング
  - 環境光 + 平行光源（太陽光相当）
  - 石表面に光沢ハイライト

### アニメーション

| アニメーション | 説明 | 時間 |
|----------------|------|------|
| 石の配置 | 上方からドロップしてバウンド | 300ms |
| 石のフリップ | Y 軸回転 180° でひっくり返る | 400ms/枚 |
| 複数フリップ | ひっくり返りを連鎖的にウェーブ演出 | 50ms ずつ遅延 |
| 配置可能ヒント | 半透明の石がゆっくり上下に浮遊 | ループ |
| ゲーム終了 | 勝利石が一斉にジャンプ | 600ms |

### シェーダー

```glsl
// 頂点シェーダー: MVP変換、法線変換
// フラグメントシェーダー: Blinn-Phong ライティング
//   - ambient + diffuse + specular
//   - 石の色は uniform で制御
```

## ゲームモード

### 対戦モード

| モード | 説明 |
|--------|------|
| 1P vs CPU | プレイヤー（黒）対 CPU（白） |
| 2P ローカル | 同一端末で2人対戦 |

### CPU 難易度

| 難易度 | アルゴリズム |
|--------|-------------|
| Easy   | ランダム着手 |
| Normal | Minimax 深さ3 |
| Hard   | Minimax + αβ 枝刈り 深さ6 |

### CPU 評価関数

- 石数差
- 角・辺の重みボーナス（位置評価テーブル）
- 着手可能数（mobility）

## ユーモア仕様（カオスモード）

設定パネルで「カオスモード」のON/OFFを切り替えられる。**デフォルトはON**。

### 1. 重力滑落

ボードをカメラ操作で傾けすぎると（仰角が ±65° を超えると）、重力に逆らえなくなった石がズルズルと端へ滑り始める。そのままにすると石がボード外に落下してゲーム終了。「ドボン」効果音とともに盤上の全石が奈落へ消える。

- 傾き角が 50°〜65° の間は石が少し震えて警告を発する
- 落下前にカメラを戻せば石は元の位置に戻る（ヒヤリハット演出）
- 落下した石の数がスコアに記録される（「粗相した石: XX枚」）

### 2. 蓄力破壊

配置マスを長押しすると力が蓄積され、プレスゲージが赤くなっていく。指を離した瞬間にそのエネルギーで周囲の石がひっくり返る。

| 蓄積時間 | 効果 |
|----------|------|
| 〜0.5秒 | 通常配置（変化なし） |
| 0.5〜1.5秒 | 周囲1マスの石をまとめてひっくり返す |
| 1.5〜3秒 | 周囲2マスを爆風でひっくり返す（ルール無視） |
| 3秒超 | ボードが「粉砕」されてゲーム終了。画面が崩壊するエフェクト再生 |

### 3. CPUの感情爆発

CPUが3手連続で不利になると「ムカ！」とキレ始め、以下の挙動を示す。

- 怒りゲージがUIに表示され、ゲージが満タンになるとCPUが盤上の石を1枚つまみ上げて投げつける（ランダムなマスに着地し、そこの石をひっくり返す）
- Hard難易度のCPUが敗北確定を検知すると「ちょっと待って！もう一回！」と宣言し、自動的に新規ゲームを強制リセットしようとする（プレイヤーが「拒否」ボタンを押すと勝利確定）
- CPUが機嫌の良いとき（3手連続で優勢）は石を丁寧に置くモーションになり、BGMのテンポが少し速くなる

### 4. くしゃみ乱流

約5%の確率で、ターン終了時にボードが「ハックション！」とくしゃみをする。

- くしゃみの震動で近くの石が2〜3マス吹き飛ぶ（落下した石は消滅）
- くしゃみの方向は毎回ランダム
- 連続してくしゃみが起きると「花粉多すぎ」の警告テキストが表示される
- 石が少ない側のプレイヤーには「くしゃみ除け」お守りアイテムが1回分支給される（次のくしゃみを無効化）

### 5. 石のサボり・反乱

連続して3ターン以上ひっくり返されていない石は「退屈」になり、以下の行動をとる。

- 退屈ゲージが溜まると石の上に「Zzz...」が浮かび、その石は着手判定から外れる（無効マス扱い）
- 6ターン以上放置されると完全に眠り込み、盤上で寝返りを打ってランダムに向きが変わる
- 眠った石の隣のマスをクリックすると石が「ビクッ！」と起きて再び有効化される（くすぐり起床）
- 10ターン以上ひっくり返された石（酷使された石）は「もう嫌だ！」と反乱を起こし、次のターンに自発的にランダムな色に変わろうとする。プレイヤーが素早くクリックすれば鎮圧できる
